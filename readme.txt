INSTALL
------------

1.Extract the zip file in a directory.
2.Create a new mysql database and give all its permissions to a mysql user.
3.Run the SQL commands found in sql.sql (use a text editor to open it) in phpmyadmin or equivalent in the created database.
4.open configs.php in a text editor.
5.Make neccessary changes in the configs.php file.
